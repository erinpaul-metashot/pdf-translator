# Sarvam Documentation Truth Pack

This folder contains scraped and normalized markdown notes from Sarvam documentation for building the PDF translator.

## Files

- `00-source-index.md` - Full sitemap URL inventory (scrape source list)
- `01-core-platform-truths.md` - Auth, models, pricing, rate limits, platform constraints
- `02-pdf-translation-api-truths.md` - API-level details for document processing + translation
- `03-cookbook-integration-truths.md` - Production patterns from cookbook/integration guides
- `04-pdf-translator-implementation-truth.md` - Implementation blueprint tailored to this project

## Notes

- Content is extracted from `https://docs.sarvam.ai` pages and normalized for engineering use.
- Citations/URLs are embedded in the truth files for traceability.
